# ♟ Шахматы

Браузерные шахматы для игры вдвоём за одним экраном (hot-seat).

## Возможности

- Полные правила шахмат: все фигуры ходят корректно
- Рокировка (короткая и длинная)
- Взятие на проходе (en passant)
- Превращение пешки с выбором фигуры
- Определение шаха, мата и пата
- Подсветка легальных ходов
- История ходов в шахматной нотации
- Отображение взятых фигур

## Как играть

1. Открой `chess.html` в любом браузере
2. Кликни на фигуру — подсветятся доступные ходы
3. Кликни на подсвеченную клетку — фигура переместится
4. Играйте по очереди: белые ходят первыми

## Запуск

Просто открой файл `chess.html` — никаких зависимостей не требуется.
